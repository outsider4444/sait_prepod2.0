Функции работы на странице :
Просмотр данных:
  1. При нажатии на кнопку "Прейти на страницу предмета" произойдёт переход на страницу предмета "МДК 02.01 Теория разработки программного обеспечения".
  2. На странице представлена таблица которая отображает список лекций или практик в зависимости от выбора пользователя.
  3. При нажатии на кнопку происходит переход на страницу где выводится таблица с оценками студентов.
  4. При нажатии на кнопку "Поиск по дате" на странице просмотра оценок происходит переход на страницу с выбором диапазона дат для вывода оценок.
Ввод данных:
  1. При нажатии на кнопку "Создать новую лекцию" или "создать новую практику" в зависимости от выбора пользователя происходит переход на страницу с добавлением файла.
  2. При нажатии на кнопку "Создать новую оценку" на странице просмотра оценок происходит переход на страницу с добавлением оценки.
Удаление даление данных:
  1. При нажатии кнопки "Удалить" происходит переход на страницу удаления файла
